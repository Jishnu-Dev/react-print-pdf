Display dynamic values based on your document, such as page numbers and running headers.

## PageNumber

Returns the current page number.

```jsx template.tsx

#### Custom Style

You can use a custom CSS counter-style, by passing a known name or a custom counter style.

<Frame type="glass"><img src="../images/previews/pagenumber-92302e6e/document.1.jpg" style={{ maxHeight: '400px', borderRadius: "0.25rem", overflow: "hidden" }} /></Frame>

<div style={{paddingTop: "1rem", paddingBottom: "1rem"}}><CodeGroup>

<PageNumber counterStyle="lower-roman" />;

```
```css base.css
@import url("https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap");

html,
body {
  font-size: 28px;
  font-family: "Inter", sans-serif;
}

@page {
  size: A4;

```
</CodeGroup></div>

## RunningH1

Returns the current page's running header of level 1.

### Examples

#### Preview

Show the current running header of level 1 in the page header. All running headers are reset when any of their parent headings are encountered (e.g. a level 2 heading resets the level 3, 4, 5 and 6 headings).

<Frame type="glass"><img src="../images/previews/runningh1-80c213c1/document.1.jpg" style={{ maxHeight: '400px', borderRadius: "0.25rem", overflow: "hidden" }} /></Frame>

<div style={{paddingTop: "1rem", paddingBottom: "1rem"}}><CodeGroup>
```jsx template.tsx
import { RunningH1 } from "@onedoc/react-print";

<React.Fragment>
  <PageTop style={{ paddingTop: "1rem" }}>
    <RunningH1 />
  </PageTop>
  <h1>Heading of level 1</h1>
</React.Fragment>;

```
```css base.css
@import url("https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap");

html,
body {
  font-size: 28px;
  font-family: "Inter", sans-serif;
}

@page {
  size: A4;
}

```
</CodeGroup></div>
